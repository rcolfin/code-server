# code-server

## About

This repo builds upon the [docker-code-server](https://github.com/linuxserver/docker-code-server) image, installing dependencies required for other development environments such as Python.

The base image is [linuxserver/code-server](https://hub.docker.com/r/linuxserver/code-server), publishing the following images:
* [rcolfin/metelsoft:code-server-python](https://hub.docker.com/repository/docker/rcolfin/metelsoft/code-server-python)

## Contributing

If interested in supporting other languages, or adding more dependencies, feel free to reach out, submit an issue, or open a pull-request.
